## Installation
Install this repository via [Package Control](http://wbond.net/sublime_packages/package_control)

## Git Clone
Clone this repository in to the Sublime Text 3 `Packages` directory, which is located where ever the `Preferences` -> `Browse Packagesâ€¦` option in Sublime Text takes you.

## Usage
To insert or change a selected color, use `ctrl+shift+c`
